----- DESCRICÃO DO PROJETO -----

Um aplicativo simples baseado em React Native e escrito na linguagem TypeScript para criar perfis de usuários e registra-los localmente

Utilidades como registro de clientes ou funcionários em um estabelecimento comercial,
controle de acesso de indivíduos a locais,
entre outros usos.

----- RODANDO O PROJETO ------

É necessário instalação do programa Expo no dispositivo e uso do terminal do Windows para rodar o arquivo root do projeto. 
Na linha de comando, ao chegar na pasta raiz, inicializar o Expo com o comando 
"npx expo start"
para inicializar o aplicativo e em sequencia selecionar a visualização "web" para roda-lo no browser default do dispositivo.
